Fast Project Docs MCP Server

Project Overview

Type: Open Source Project
Repository: GitHub - fast-project-docs-mcp
License: MIT

Challenge

Creating comprehensive project documentation (PRDs, technical specifications, and design documents) is time-consuming and often inconsistent. Product managers and developers spend hours writing documentation that could be automated while maintaining professional quality and structure.

Solution & Implementation

Built a Python-based Model Context Protocol (MCP) server that integrates with AI clients like Claude Desktop to automate the generation of professional project documentation. Inspired by Kiro's spec-driven development methodology, the system uses structured templates and intelligent prompts to create comprehensive documents from user input and reference materials.

Key Features Delivered

  • Multi-Document Generation: Automated creation of PRD (Product Requirements Document), SPEC (Technical Specification), and DESIGN documents
  • Reference Resource Analysis: Processes multiple file formats (Markdown, JSON, YAML, PDF, images) to incorporate existing project materials
  • MCP Protocol Integration: Full Model Context Protocol support for seamless integration with AI clients
  • Template Customization: Flexible template system following industry best practices with customization options
  • Cross-Document References: Maintains consistency and relationships between related documents

Technical Architecture

Core Technologies: Python 3.9+, MCP Framework, AsyncIO
File Processing: PyPDF2, python-docx, Pillow, pytesseract (OCR)
Data Processing: PyYAML, python-frontmatter, markdown
Development Tools: pytest, black, mypy, flake8

Impact & Results

  • 2,000-4,000+ word documents generated automatically with professional structure
  • Multi-format support for reference materials including PDFs and images with OCR
  • Open source contribution to the MCP ecosystem with MIT license
  • Comprehensive testing with unit and integration test coverage
  • Production-ready with security scanning, code quality tools, and documentation

Documentation Generated

The system generates three types of professional documents:

  • PRD: Product requirements with user stories, success criteria, and business constraints
  • SPEC: Technical specifications with architecture, APIs, and implementation details
  • DESIGN: Design documents with system architecture, data flows, and UI specifications

RAG-Powered Meal Planner

Project Overview

Type: Personal Project
Repository: GitHub - meal-planner-RAG
License: MIT

Challenge

Meal planning and grocery shopping are often disconnected processes, leading to forgotten items, repetitive meals, and manual tracking of preferences. This project aimed to create an intelligent assistant that learns from user history to provide personalized and efficient meal and shopping suggestions.

Solution & Implementation

A CLI application that uses Retrieval-Augmented Generation (RAG) to offer personalized meal plans and smart shopping lists. The system leverages a FAISS vector database and sentence transformers to analyze the user's cooking history, preferences, and past shopping bills, providing tailored recommendations through an AI-powered interface.

Key Features Delivered

  • Personalized Meal Planning: Generates three meal plan options based on cooking history and preferences.
  • Smart Shopping Lists: Creates comprehensive shopping lists including both food and household items.
  • RAG-Powered Engine: Utilizes FAISS and sentence transformers for intelligent context retrieval from user data.
  • Learning System: Tracks user choices to continuously improve future suggestions.
  • Interactive Data Entry: Offers a user-friendly CLI for adding meals, bills, and grocery lists.

Technical Architecture

Core Technologies: Python, RAG, FAISS, Sentence Transformers
AI Integration: OpenRouter for LLM access (Claude 3.5 Sonnet)
CLI: Click and Rich for an interactive command-line experience
Database: FAISS for vector similarity search

Impact & Results

  • A fully functional CLI application for personal meal and shopping management.
  • Demonstrates practical application of RAG for personalization.
  • Open-source and available for others to use and contribute.
