Inventory System Integration

Project Overview

Role: Technical Product Manager
Company: LabTwin GmbH (acquired by Labforward)
Duration: January 2023 - November 2024

Challenge

Scientists using Revvity Signals Notebook needed a seamless way to manage laboratory inventory without interrupting their experimental workflows. Traditional inventory management required manual data entry, disrupting sterile environments and taking focus away from critical research tasks.

Solution & Implementation

Extended the existing LabTwin voice assistant integration with Revvity Signals Notebook to include comprehensive inventory management capabilities. Built a unified API layer that synchronized voice commands with inventory database updates in real-time.

Key Features Delivered

  • Scan & Locate: Voice-activated item scanning with location tracking and mobile device access to critical information
  • Check In/Out: Seamless inventory status updates without workflow interruption
  • Real-Time Voice Updates: Hands-free maintenance of accurate, cloud-connected inventory records
  • Unified Data Capture: Combined experimental data and inventory management in a single voice-driven workflow

Technologies Used

Voice Recognition • API Integration • Cloud Connectivity • Mobile Development • Laboratory Information Systems • Real-time Data Synchronization

Impact

The integration created a fully connected digital lab experience, minimizing manual data entry, supporting compliance requirements, and empowering researchers to focus on breakthrough innovations rather than routine inventory tasks. This collaboration between Labforward and Revvity Signals represents a new era of connected laboratory workflows.

Thryve Health Data API

Project Overview

Role: Product Owner
Company: Thryve GmbH
Duration: October 2019 - December 2022

Challenge

Health industry companies struggled with fragmented data sources and complex integrations when building health applications. Each wearable device required separate integration efforts, creating 3-6 months of development overhead per device.

Solution & Implementation

Built a unified SDK that provided seamless access to multiple health data sources through a single API. Developed standardized data schemas and handled device-specific complexities behind a clean, developer-friendly interface.

Technical Implementation

  • Data Normalization Engine: Built ETL pipeline processing 10M+ data points daily
  • Real-Time Webhooks: Built the event-driven architecture for instant data updates
  • Multi-Tenant Architecture: Designed scalable system supporting 100+ concurrent customers
  • Device Integration Layer: Created abstraction layer handling 10 major wearable APIs

Business Impact

  • 1M+ users served across the platforms
  • €1M ARR contributed to company growth
  • 100+ customers across insurance, digital health, and academic research
  • 10 major integrations including Oura Ring, Whoop, Omron Wellness, Samsung Health
