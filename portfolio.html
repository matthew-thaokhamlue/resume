LabTwin AI Integration

Project Overview

Role: Technical Product Manager
Company: LabTwin GmbH (acquired by Labforward)
Duration: January 2023 - November 2024
Team Size: 8 members (6 engineers, 1 designer, 1 customer support)

Challenge

Scientific laboratories needed a hands-free solution for documentation and data entry during experiments. Traditional methods required researchers to stop their work, remove gloves, and manually input data, disrupting workflow and potentially compromising sterile environments.

Solution

Led the integration of advanced NLP and speech-to-text models specifically trained for scientific terminology and laboratory environments. The solution enabled researchers to document experiments, record measurements, and manage inventory through voice commands.

Key Achievements

  • 95% accuracy in scientific domain speech recognition
  • 2 major integrations delivered: Revvity Signals inventory management and LabVantage LIMS
  • 12 releases per year increased from 4 through improved delivery process

Technologies Used

Python • Natural Language Processing • Speech-to-Text Models • API Integrations • Machine Learning • Laboratory Information Management Systems (LIMS)

Impact

The AI integration transformed how researchers interact with laboratory systems, enabling seamless voice-driven workflows that maintain sterile environments while improving data accuracy and research efficiency.

Thryve Health Data API

Project Overview

Role: Product Owner
Company: Thryve GmbH
Duration: October 2019 - December 2022
Team Growth: Scaled from 7 to 35 members across 7 specialized teams

Challenge

Health industry companies struggled with fragmented data sources and complex integrations when building health applications. Each wearable device and health platform required separate integration efforts, creating significant development overhead.

Solution

Built a unified SDK that provided seamless access to multiple health data sources through a single API. The platform standardized data formats and handled the complexity of individual device integrations behind a clean, developer-friendly interface.

Key Achievements

  • 1M+ users served across the platform
  • 100+ customers across insurance, digital health, and academic research
  • €1M ARR contributed to company growth
  • 10 major integrations including Oura Ring, Whoop, Omron Wellness, Samsung Health

Technologies Used

SDK Development • RESTful APIs • Data Integration • Health Data Standards • Wearable Device APIs • Cloud Infrastructure • Data Analytics

Impact

The SDK became the foundation for numerous health applications, enabling insurance companies, digital health startups, and research institutions to build data-driven solutions without the complexity of managing multiple device integrations.

Automation Tools

Project Overview

Type: Weekend Side Projects
Focus: Personal Productivity & Workflow Automation
Duration: Ongoing (2024 - Present)
Tech Stack: Python, Gemini, Claude, OpenAI

Challenge

Repetitive manual tasks in product management workflows were eating up valuable time that could be spent on strategic work. Creating release notes, writing Jira tickets, and API testing required consistent formatting and attention to detail but offered little creative value.

Solutions Built

Developed a suite of Python-based automation tools that leverage LLMs and APIs to streamline common product management tasks, turning hours of manual work into minutes of automated execution.

Tools Developed

  • Release Notes Generator: Python app with LLM integration that transforms raw feature lists into polished product release notes
  • Jira Ticket Creator: Text-to-ticket tool that converts natural language input into properly formatted Jira bugs and user stories
  • API Tester: Developed a round-trip testing script for comprehensive API validation and regression testing

Technical Approach

Python Development • LLM API Integration • REST API Testing • Natural Language Processing • Automated Documentation • CLI Tools

Impact

These self-built tools have significantly reduced time spent on routine tasks, allowing more focus on strategic product decisions and user experience improvements. Each tool addresses a real pain point in daily workflows.
